 # PyEncryptionApp
This app is built on Ubuntu Linux for encrypting and decrypting file/folder.

# Language
Python 3

# Algorithm
3DES, SHA-160, MD5

# Some packages you should install
```
pip install crypto
```
```
pip install hashlib
```
```
sudo apt-get install python-tk
```
